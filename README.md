# csv2json-app

A CSV ↔ JSON converter built with HTML, CSS, and JavaScript.

## Features

- [x] Paste CSV and convert to JSON
- [x] Paste JSON and convert to CSV
- [x] Warning message for empty or invalid input
- [x] Clear button to reset both text boxes
- [x] **Bonus:** Open CSV/JSON file from local file system
- [x] **Bonus:** Save CSV/JSON output to a file

## Tech Stack

- HTML
- CSS
- JavaScript (vanilla)

## How to run

Open `index.html` in your browser — no build tools needed.
